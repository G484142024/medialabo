数当てゲーム

課題3-1では、JavaScriptの基本的な文法を使って、1から10までのランダムな数字を当てる数当てゲームを作成しました。 ユーザーは最大3回まで予想を入力でき、予想した数字が正解かどうかだけでなく、「もっと大きい」「もっと小さい」といったヒントもコンソールに表示されるようにしています。 また、もしユーザーが最初の予想で正解した場合は、次の予想入力時にゲームが終了したことを表示するようにしています。

各授業愛の学習事項

第1回 WWW と HTML

今回、HTMLのいくつかの要素を学んで HTMLによるWebページの構造を作りました。Webページの構造は様々な要素を使ってより良いページが作れました。 要素の例として、ハイパーリンクや画像の埋め込み、説明リストなどの要素を学びました。また、headやbodyの要素のより、ページの構造が整えるようになるということがわかりました。

演習の成果:　演習1-4:HTMLのさまざまな要素

第2回 CSS による WEBページの装飾

今回、CSSを使ってHTMLの見た目をより整えて綺麗にするという目的として学びました。CSSには様々なプロパティがあって自分で調べなければならないということも分かりました。 このプロパティを利用して、Webページの装飾になれると分かりました。例えば、背景やリンクの色を変えたりborder,fontやpaddingなどを付けたりすることでWebページがWebページっぽいに見えるようになりました。

演習の成果:　演習2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

今回、JavaScriptの言語を学んでWebページ上で様々な機能や動きを実現できるということがわかりました。Javaと異なりJavaScriptの方がコンパイラを使わずに1行ずつ実行させる言語です。 また、HTML ファイルとは別に JavaScript ファイルを作成するやり方があってsrcとdeferという属性を書かなければなりません。

演習の成果:　演習3-6: オブジェクトの参照
